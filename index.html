projects

The 'Path-Finding Visualizer' is a project that demonstrates how the Dijkstra algorithm is used to find the shortest path between two points on a map. The visualizer shows the algorithm in action, starting with the initial positions of the source and destination points, and then iteratively updating the distances and paths of the nodes until the shortest path is found. The visualizer allows users to see how the algorithm works and how it finds the optimal path in a visually appealing way. This project can be useful for students and researchers who are learning about graph algorithms and path finding, as well as for anyone who is interested in the inner workings of the Dijkstra algorithm.


Technologies used: Pygame, Python
